# Simple Text Editor (Notepad)

## 📖 Overview

This is a lightweight text editor developed using Python's Tkinter library. It allows users to perform essential text editing tasks such as creating new files, opening existing files, and saving content.

## 🚀 Features

* Create a new text file
* Open and read `.txt` files
* Save text to a file
* Simple and clean GUI
* Dark theme (black background with yellow text)
* Menu-based navigation

## 🛠️ Technologies Used

* Python 3
* Tkinter (built-in GUI library)

## 📂 File Operations

* **New:** Clears the text area
* **Open:** Loads content from a `.txt` file
* **Save:** Saves current text into a file
* **Exit:** Closes the application

## ▶️ How to Run

1. Make sure Python is installed on your system.
2. Save the script as `text_editor.py`
3. Run the file:

   ```bash
   python text_editor.py
   ```

## 📌 Requirements

* Python 3.x (Tkinter comes pre-installed)
